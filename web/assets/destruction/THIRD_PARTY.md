# Third-party runtime dependencies

The generated GLB assets, textures and integration code are original and contain no third-party logos or branded geometry.

The demo and integration layer expect these external libraries, which are not redistributed in this package:

- Three.js 0.185.1 — rendering and GLTFLoader.
- @dimforge/rapier3d-compat 0.19.3 — WebAssembly rigid-body physics.

Those projects retain their respective licenses. Review them when deploying the dependencies in your application.
